1-Trocar uma lâmpada:

1- Posicionar escada no centro da sala.2Subir com cuidado.
2- Desenroscar lâmpada queimada.
3- Enroscar nova lâmpada.
4- Testar funcionamento.


2-Ir ao mercado:

1- Caminhar até o mercado.
2- Seguir lista de compras.
3- Escolher produtos.
4- Pagar no caixa.
5- Retornar para casa.


3-Arrumar uma mala:

1- Colocar mala aberta sobre a cama.
2- Dobrar roupas.
3- Organizar sapatos em sacos.
4- Reservar espaço para itens de higiene.
5- Fechar mala.


4-Escovar os dentes:

1- Pegar escova.
2- Aplicar pasta de dente.
3- Movimentar em círculos nos dentes.
4- Enxaguar boca e escova.


5-Chegar até a escola:

1- Se preparar.
2- Conferir se não esqueceu nada.
3- Sair de casa com mochila.
4- Caminhar pela rua.
5- Cumprimentar vizinhos.
6- Chegar à escola.


6-omprar um remédio:

1- Dirigir-se à farmácia.
2- Solicitar medicamento ao farmacêutico.
3- Discutir dosagem.
4- Pagar pelo remédio.
5- Retornar para casa.


7-Encher a garrafa de água:

1- Pegar garrafa vazia.
2- Ir até o bebedouro.
3- Inclinar garrafa.
4- Pressionar botão para encher.
5- Fechar garrafa.


8-Amarrar os sapatos:

1- Sentar-se.
2- Cruzar cadarços.
3- Fazer laço básico.
4- Formar duas alças.
5- Cruzar e puxar para nó firme.


9-Upload no GitHub pelo GitHub Desktop:

1- Abrir GitHub Desktop.
2- Selecionar repositório.
3- Arrastar arquivo para interface.
4- Deixar mensagem de commit.
5- Clicar em 'Commit to master'.
6- Clicar em 'Publish repository'.


10-Enviar anexo no Classroom:

1- Abrir plataforma do Classroom.
2- Selecionar atividade.
3- Clicar no ícone de clipe.
4- Buscar arquivo nos documentos.
5- Confirmar envio de anexo.
6-Submeter atividade.
